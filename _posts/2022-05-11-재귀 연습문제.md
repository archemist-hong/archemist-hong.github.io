```python
def get_binomial_coefficient(n, k): # n combination k(nCk)
    if k == 0 or k == n:
        return 1
    elif 0 < k < n:
        return get_binomial_coefficient(n-1, k-1) + get_binomial_coefficient(n-1, k)
    else:
        print('Input Error')
```

**반복**

```python
def get_binomial_coefficient(n, k):
    bc = [[1] * (n+1) for _ in range(n+1)]
    if k == 0 or k == n:
        return 1
    elif 0 < k < n:
        for i in range(n):
            for j in range(i):
                bc[i+1][j+1] = bc[i][j] + bc[i][j+1]
        return bc[n][k]
```



## 18.

**재귀**

```python
def Ackermann(m, n):
    if m == 0:
        return n + 1
    elif n == 0:
        return Ackermann(m-1, 1)
    else:
        return Ackermann(m-1, Ackermann(m, n-1))
```

> 반복으로는 아직 풀지 못했다. 재귀 -> 반복으로 고치는 방법을 좀더 공부해야겠다.

```python
def write_pixel(x, y, color):
    global screen
    screen[x][y] = color

def flood_fill(x, y):
    if read_pixel(x, y) == 0:
        write_pixel(x, y, 2)
        flood_fill(x + 1, y)
        flood_fill(x - 1, y)
        flood_fill(x, y + 1)
        flood_fill(x, y - 1)
```
