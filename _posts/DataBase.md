# DB와 SQL



## DB의 정의

DB : 데이터의 집합

DBMS : DB를 관리하는 SW

DBMS의 조건

- 대용량 데이터 처리
- 공유 가능(여러명이 동시작업)



## DBMS의 종류

**MySQL**,  MariaDB, Oracle, SQL server 등

완전한 무료 사용을 원한다면 Maria DB 사용 권장



## SQL 정의

SQL :  DBMS에 데이터를 구축, 관리하고 활용하기 위해서 사용되는 언어

표준 SQL : 국제 표준 SQL (DBMS의 종류에 관계없이 동일함)



## DB의 종류

- 계층형 DB : 트리형태로 계층적 구조
- 망형 DB : 계층적 구조에서 서로 연결 가능
- 관계형 DB : 테이블(행x열)로 구성되며, 테이블 사이의 관계를 모델링
